How to Install

Gmail: Settings → See all settings → General → Signature → click the code editor icon (</>) → paste the copied HTML.

Outlook: Settings → Mail → Compose and reply → Email signature → use source editor to paste HTML.

Apple Mail: Mail → Settings → Signatures → create new → paste from a browser preview (not raw HTML).
